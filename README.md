# Energy Price Prediction Dashboard

This project predicts Germany energy prices using machine learning models.

## Models
- Linear Regression
- Random Forest Regressor

## Features
- Hour
- Day
- Month
- Day of Week
- Year
- Previous Germany price
- Germany price 24 hours earlier
- France, Italy, Spain, and UK energy prices

## Results
- Linear Regression R2 Score: 0.971
- Random Forest R2 Score: 0.976

## Tools Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib
